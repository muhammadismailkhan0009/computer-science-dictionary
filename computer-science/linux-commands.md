# Linux Commands

## Word: man
### Description:
"man" command is linux command used to provide the user manual of any command of linux that is given as its argument.
The simplest syntax of the command is:
'''cmd
$ man command_name
'''
For example, to find manual of "host" command:
'''cmd
$ man host
'''
